## Compatibility
- PHP (8.0)(https://reintech.io/blog/installing-php-8-on-ubuntu-22)
- Composer (2.0)
- Lumen (10.0)
- Apache or Nginx
- Mysql

## Setup
> Clone the repo
```sh
$ git clone https://github.com/sagnikcapital/Google-Drive-Up-loader.git
```
> Create .env file
```sh
$ cp .env.example .env
```
> Composer install
```sh
$ composer install
```
### Custom make Job command
```sh
$ php artisan make:job SampleJob
```
